# Mr.Pitani Storefront

React + TypeScript storefront for browsing frozen food products, placing retail orders, and submitting bulk enquiries.

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS + shadcn/ui
- React Query
- Supabase
- Vitest

## Local Development

```sh
npm install
npm run dev
```

## Quality Checks

```sh
npm run test
npm run lint
```

## Build

```sh
npm run build
npm run preview
```
